Data Generator Form

The User Interface (UI) offers the following selection of input:

  • Data-type(s) [ Name, Gender, Country, Race, Address, Credit Card ]
  • Number of records [ range from 0 to 1000 ]
  • Statistic distribution [ optional ]

Statistic distribution field

Gender

User can indicate the percentage as provided in the textbox. Further validation is provided for the sum of Male & Female percentage. [ Total of 100% ].

Country | Race | Credit Card

User are limited to 2 or lesser filter of distribution.

Address

The program has no application for this field statistical distribution as Address is a subset of Country.
Example of selection Result panel for generated data

Data Generator supports the following 2 type of display:

Table

In the Table display view, data generated are shown in a table.

SQL

In the SQL display view, data generated are shown as SQL DDL CREATE & DML INSERT statement.
